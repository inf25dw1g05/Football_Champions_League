# C1 : Champions League - Desenvolvimento Web I

## Description

Ao longo deste segundo momento de avaliação da cadeira de Desenvolvimento Web o nosso trabalho baseou-se no desenvolvimento de serviços de uma API REST com a utilização dos métodos CRUD (Create, Read, Update, Delete), para gerir informação de uma fase de grupos da Liga dos Campeões.  
A aplicação inclui o backend, a ligação à base de dados e toda a gestão dos dados relevantes: equipas, jogadores, grupos, jogos e eventos de jogo dentro desses mesmos jogos (exemplo : golos, cartões amarelos e vermelhos).

A API foi construída segundo um ficheiro `openapi.yaml` que define todas as rotas e modelos de dados.  
A arquitetura baseia-se em controladores, serviços e modelos, permitindo uma estrutura organizada, reutilizável e de fácil manutenção.

Para garantir portabilidade e facilidade de execução, o projeto utiliza um ambiente **multicontainer Docker**, com containers para:
- API;
- Base de dados;

Com `docker compose up`, o sistema arranca automaticamente, configurando a base de dados, levantando a API e disponibilizando o serviço na porta definida.
